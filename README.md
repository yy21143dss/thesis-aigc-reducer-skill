# thesis-aigc-reducer

面向本科毕业论文的 **Cursor Agent Skill**：在**不改变核心观点与事实**的前提下，协助调整表述以降低 **AIGC 检测**或**传统查重**风险；支持 **LaTeX** 与**纯文本**，建议**逐段**处理。

（本仓库在 GitHub 上名为 `thesis-aigc-reducer-skill`；安装到 Cursor 时文件夹名仍建议使用 `thesis-aigc-reducer`，与 `SKILL.md` 中 `name` 字段一致。）

## 功能概览

- **AIGC 降率模式**：针对知网、维普等 AIGC 检测场景，通过句式重组、结构调整和表达多样化来弱化 AI 生成特征，**严格保持学术论文的正式书面语体**，绝不以口语化方式降低检测率。
- **传统降重模式**：侧重同义替换、句式变换与语序调整，在语义不变的前提下降低与已有文献的连续重复风险。
- **LaTeX**：解析结构后只改写可替换正文，尽量保留 `\cite`、`\ref`、公式与图表相关标记。
- **输出**：纯文本为「改写结果 + 修改说明」；LaTeX 输入为「LaTeX + 纯文本预览 + 修改说明」。

## 最新改动说明（2026-05-12）

- **改写后质量自查（语病检测）**：每次改写完成后、输出前须逐句自查语法与搭配、用词规范、语义是否走样、段落整体是否像合格论文正文；未通过须先修正再输出。LaTeX 流程已增加与之对应的「步骤五」。
- **禁止口语化降 AIGC**：明确不得以口语、语气词或随意短句换「人味」；弱化 AI 特征应靠句式与论述节奏，**必须保持正式书面语**。
- **语法与生造词约束**：强调语法正确性与学术正式性同等重要，禁止生造词与不规范搭配；若改写后无法保证通顺，应保留更接近原文的写法。
- 完整条目见 [CHANGELOG.md](CHANGELOG.md)。

## 适用对象

- 已自行完成论文写作，希望对段落做**合规范围内的语言润色与结构调整**的同学。
- 使用 Cursor，并希望通过 `@` 引用 Skill 统一改写流程的用户。

## 安装方式

将本仓库整个文件夹放到以下任一位置（文件夹名保持 `thesis-aigc-reducer`）：

| 类型 | 路径（示例） |
|------|----------------|
| 项目级 | `<你的项目>/.cursor/skills/thesis-aigc-reducer/` |
| 个人级 | `~/.cursor/skills/thesis-aigc-reducer/`（Windows 多为 `C:\Users\<用户名>\.cursor\skills\thesis-aigc-reducer\`） |

目录内应至少包含：`SKILL.md`、`strategies.md`、`examples.md`。

## 使用方法

1. 在 Cursor 对话中 `@thesis-aigc-reducer`（或你安装后的 Skill 名称）。
2. 说明模式，例如：**「帮我降 AIGC」** 或 **「帮我降重」**；若未说明，Skill 会请你二选一。
3. **逐段**粘贴需要改写的内容（不宜一次粘贴整篇过长文本）；可说明是否为 LaTeX。
4. 对某句不满意时，可指出位置要求**局部再改**，支持多轮迭代。

详细流程与约束见仓库内 `SKILL.md`。
