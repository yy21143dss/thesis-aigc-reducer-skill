# 改写策略详细参考

本文件为 thesis-aigc-reducer skill 的改写策略参考。agent 在执行改写时应读取本文件并综合运用相关策略。

---

## AIGC降率策略

目标：在**严格保持学术论文正式书面语体**的前提下，通过结构重组和表达多样化消除 AI 生成文本的规律性特征。

**最高原则：绝不以口语化来降低 AIGC 率。** 改写后的文本必须仍然是合格的学术论文语言，能直接用于毕业论文正文、摘要等正式章节。

### 改写禁忌清单

以下是已知的典型错误模式，改写时**严禁**出现：

| 错误类型 | 错误示例 | 问题所在 | 正确做法 |
|---|---|---|---|
| 搭配不当 | "难以先验固定的环节" | "环节"不能用"固定"修饰，"先验固定"是生造词 | "难以事先确定的因素" |
| 生造词语 | "屏障衰减" | "屏障"与"衰减"搭配不当，非规范学术用语 | "免疫保护效力的衰减" |
| 不当比喻 | "复制天花式的根除叙事" | "根除叙事"非学术用语，过度文学化 | "实现类似天花的彻底根除" |
| 口语表达 | "把易感池'补回去'" | "补回去"是口语 | "使易感人群比例重新上升" |
| 非规范搭配 | "落入易感者集合" | "落入集合"不符合学术表达习惯 | "重新转变为易感个体" |
| 口语化连接 | "不过话说回来" | 口语化转折 | "但需要指出的是" |
| 随意短句 | "结果很明显。" | 过于随意，不符合学术论文语体 | "实验结果较为显著。" |

**核心原则：宁可保留更接近原文的表达，也不要为了打乱句式而制造语病或不规范用语。**

### 1. 句式重组

AI 生成的文本倾向于使用结构整齐、语法完美的句子。需要打破这种规律性，但**仍须使用正式书面表达**。

**具体手法：**

- **长句拆短**：将一个包含多层修饰的长句拆为 2-3 个短句，每个短句仍为正式学术表达
  - 原句："随着深度学习技术的不断发展，越来越多的研究者开始将其应用于自然语言处理领域，并取得了显著的效果。"
  - 改写："深度学习技术近年来发展迅速。已有大量研究将其应用于自然语言处理领域，并取得了较为理想的成效。"

- **短句合并**：将几个碎片短句合并为一个结构更丰富的复杂句
  - 原句："该方法效果好。它计算速度快。它适用范围广。"
  - 改写："该方法不仅在预测效果上具有优势，在计算效率与适用范围方面亦表现突出。"

- **因果倒置**：将结论/结果放在句首，原因/条件放在句尾
  - 原句："由于样本数量有限，实验结果可能存在一定偏差。"
  - 改写："实验结果可能存在一定偏差，其主要原因在于样本数量较为有限。"

- **插入语/同位语**：在句子中间插入限定或补充成分，打破句子的线性流畅感
  - 原句："神经网络模型在图像分类任务中表现优异。"
  - 改写："神经网络模型——特别是深度卷积网络——在图像分类任务中表现较为突出。"

### 2. 学术主体性表达

AI 文本通常缺乏研究者的主体视角。在学术写作中，可以通过规范的作者立场表达来体现"人的痕迹"，但**必须使用学术论文中常见的正式措辞**。

**中文学术主体性表达（按使用场景分类）：**

引出研究行为：
- "本文探讨了""本研究关注的是""本文着重分析了"
- "笔者认为""笔者注意到""据笔者考察"

分析过渡：
- "经分析可知""经对比发现""对数据进一步考察后可知"
- "从以上分析中不难看出""结合实验数据可以发现"

强调与评价：
- "值得注意的是""需要指出的是""不容忽视的一点在于"
- "尤为关键的是""有必要强调的是"

概括归纳：
- "总体而言""就当前研究现状来看""从整体上看"
- "综合上述分析""归纳而言"

**英文学术主体性表达：**

- "It is worth noting that"、"It should be emphasized that"
- "As can be observed from the results"、"A closer examination reveals that"
- "One plausible explanation is that"、"Of particular interest is"
- "To the best of our knowledge"、"The present study suggests that"

**使用原则：**
- 每段使用 1-2 个即可，不要每句都加
- 分散在段首、段中、段尾，不要集中使用
- 所有表达必须是学术论文中常见的规范措辞，禁止使用口语化表达

### 3. 句长节奏调整

AI 文本的典型特征是句长均匀，大部分句子在 20-30 字之间，几乎没有极短或极长的句子。

**调整策略：**

- 在段落中制造句长差异：混合使用较短的简明句和较长的复合句
- 在合适的位置使用精炼的判断句作为承上启下（如"这一结论值得进一步验证。"）
- 避免连续 3 句以上句长相近
- 短句仍须保持书面语体，不可使用碎片化的口语短句

**示范：**
- AI 风格（句长均匀）："该模型在准确率方面达到了较高水平。同时在召回率方面也有不错的表现。此外模型的训练速度相比之前有了明显提升。"
- 改写风格（长短交替，仍为正式学术语体）："模型准确率较高。在召回率方面，经过多轮参数调优后同样取得了较为理想的结果，但这一过程中也暴露出超参数选择对性能的敏感性问题。训练效率的提升则较为显著。"

### 4. 段落结构调整

AI 几乎总是采用"总述 → 分点论述 → 总结"的三段式结构。

**调整策略：**

- **从具体切入**：跳过笼统的总述句，直接从一个具体事实、数据或现象开始
  - AI 风格："本节将从三个方面分析该方法的优势。首先..."
  - 改写："实验结果显示，该方法在精度指标上相较基线模型有明显提升..."

- **弱化模板式总结**：段尾不必每次都使用"综上所述"，可用更具体的论断收束
  - 将"综上所述，该方法具有明显优势"改为更具体的表述，如"上述实验数据在一定程度上验证了该方法的有效性"

- **适当插入学术性的限定或讨论**：在论述中间加入条件限定或不足之处的讨论
  - 例如："需要指出的是，上述结论的成立依赖于若干前提假设""当然，该方法在某些极端情形下仍存在局限"

- **非对称分点**：如果有三个分论点，不必平均分配篇幅，可以对重点论点做更充分的展开

### 5. 逻辑连接词多样化

AI 对"首先、其次、再次、最后、因此、然而、此外、同时"等连接词高度依赖且使用模式固定。

**替换策略（均为正式学术用语）：**

| AI 偏爱的连接词 | 学术化替代方式 |
|---|---|
| 首先 | 直接陈述，不加连接词；或"就...而言" |
| 其次 | "在另一方面""从...角度来看" |
| 再次 | "除上述因素外"、或直接省略 |
| 最后 | "此外尚需关注的是""另一个不可忽视的因素在于" |
| 因此 | "基于此""由此可以认为""这一结果表明"、或直接省略 |
| 然而 | "但需要注意的是""尽管如此""与此形成对比的是" |
| 此外 | "在此基础上""另需说明的是" |
| 同时 | "与此同时"、"在...的同时"、或直接用逗号连接 |

**原则：**
- 不必替换所有连接词，保留 30-40% 即可
- 有些地方直接删除连接词，用句意本身的逻辑关系来衔接
- 替代词必须是学术论文中常见的正式用语，**禁止使用口语化连接词**（如"不过""话说回来""所以说"等）

## 传统降重策略

目标：在保持语义完全不变的前提下，彻底改变文本的表达方式，确保连续 13 字以上不与原文或他文重复。

### 1. 同义替换

替换非专业术语的词汇，保持语义不变。

**常用动词替换表：**

| 原词 | 可替换为 |
|---|---|
| 研究 | 探讨、探究、考察、分析 |
| 分析 | 剖析、解析、审视、考量 |
| 提出 | 指出、给出、提供、阐述 |
| 表明 | 显示、揭示、说明、体现 |
| 采用 | 运用、使用、借助、利用 |
| 实现 | 达成、完成、做到、达到 |
| 具有 | 拥有、带有、存在、包含 |
| 促进 | 推动、助力、带动、加速 |
| 导致 | 引起、造成、引发、带来 |
| 认为 | 指出、主张、强调、阐明 |

**常用名词替换表：**

| 原词 | 可替换为 |
|---|---|
| 方法 | 手段、途径、方式、路径 |
| 问题 | 难题、议题、困境、挑战 |
| 特点 | 特征、属性、特性、优势 |
| 领域 | 方向、范畴、层面、维度 |
| 效果 | 成效、结果、表现、成果 |
| 影响 | 作用、效应、冲击、波及 |
| 基础 | 根基、前提、根本、底层 |
| 趋势 | 走向、态势、动向、倾向 |

**常用形容词替换表：**

| 原词 | 可替换为 |
|---|---|
| 重要 | 关键、核心、不可或缺、举足轻重 |
| 明显 | 显著、突出、清晰、可观 |
| 广泛 | 普遍、大范围、全面、多方面 |
| 复杂 | 繁杂、多元、纷繁、错综 |
| 有效 | 高效、可靠、切实、行之有效 |

**注意事项：**
- 专业术语绝对不可替换
- 替换后需检查搭配是否通顺
- 同一段落中同一个替换词不要重复使用

### 2. 句式变换

**主动 ↔ 被动：**
- "本文提出了一种新方法" → "一种新方法在本文中被提出"/"本文中提出了一种新的方法"
- "研究者发现了该现象" → "该现象被研究者所发现"/"该现象在研究过程中得以发现"

**肯定 ↔ 双重否定：**
- "十分重要" → "不可忽视"/"不容小觑"
- "效果很好" → "效果并非不理想"/"效果不可谓不佳"
- "完全一致" → "并无差异"/"没有出现不一致的情况"

**陈述 ↔ 设问/反问：**
- "该方法效率高" → "该方法效率如何？从实验数据来看，效率是比较高的"
- "这个问题值得关注" → "为什么要关注这个问题？原因在于..."

**把字句 ↔ 被字句 ↔ 一般陈述：**
- "实验证明了该理论" → "该理论被实验所证明" → "通过实验，该理论获得了证明"

### 3. 拆句与合句

**长句拆分：**
- 原句："在当前大数据背景下，如何利用海量数据来训练高效的深度学习模型是学术界和工业界共同关注的核心问题。"
- 拆分："当前正处于大数据时代。如何借助海量数据训练出高效的深度学习模型，已经成为学术界与工业界都在关注的一个核心议题。"

**短句合并：**
- 原句："该算法速度快。该算法精度高。该算法易于部署。"
- 合并："该算法兼具速度快、精度高的优点，并且在实际部署方面也比较便捷。"

**句子成分重排：**
- 原句："在图像识别领域，卷积神经网络取得了突破性进展。"
- 重排："卷积神经网络在图像识别这一领域中，已经取得了突破性的进展。"

### 4. 语序调整

**因果倒置：**
- "因为数据量不足，所以模型泛化能力较差" → "模型泛化能力不太理想，究其原因，主要是数据量不够充足"

**条件后置：**
- "在学习率设置为0.01的条件下，模型收敛速度最快" → "模型收敛速度最快的情况，出现在学习率设为0.01时"

**时间/地点状语位移：**
- "近年来，自然语言处理技术发展迅速" → "自然语言处理技术在近些年发展得相当迅速"
- "在该实验中，准确率达到了95%" → "准确率在该实验中达到了95%的水平"

**定语/状语位置调整：**
- "具有高精度的检测算法" → "检测算法精度较高"/"一种精度比较高的检测算法"
